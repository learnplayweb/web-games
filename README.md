# web-games
Web Games

웹에서 실행되는 학습용 미니게임 모음입니다.

각 게임은 HTML, CSS, JavaScript를 기반으로 개발하며, 모바일 환경을 우선 고려합니다.

## Games

### Clock Game

아날로그 시계를 읽는 능력을 반복 연습하는 게임입니다.

개발 예정 기능

* 시·분·초 읽기
* 점수 및 콤보 시스템
* 모바일 입력패드 지원
* PC 키보드 입력 지원
* 캐릭터 성장 시스템
