# Clock Game

## 게임 목표

시계를 보고 올바른 시각을 입력한다.

---

## 난이도 구성

### Level 1

* 시만 학습
* 입력: 시
* 문제 수: 5

### Level 2

* 시와 30분
* 입력: 시, 분

### Level 3

* 5분 단위
* 입력: 시, 분

### Level 4

* 1분 단위
* 입력: 시, 분

---

## 입력 규칙

* 최초 선택 영역은 시이다.
* 숫자 입력 시 현재 선택 영역에 입력한다.
* 자동으로 다음 입력 영역으로 이동하지 않는다.
* → 버튼으로 다음 입력 영역으로 이동한다.
* 마지막 입력 영역에서 →를 누르면 제출한다.
* ← 버튼은 현재 입력을 한 자리씩 삭제한다.
* 현재 입력이 비어 있으면 이전 입력 영역으로 이동한다.
* 모든 입력은 숫자로 관리한다.
* 정답 비교 시 앞의 0은 구분하지 않는다.

---

## 정답 판정

* 시와 분(필요 시 초)을 모두 비교한다.
* 모두 일치하면 정답이다.
* 하나라도 다르면 오답이다.

---

## 정답 처리

* 입력 영역을 600ms 동안 초록색으로 표시한다.
* 문제당 Gold를 지급한다.??
* Combo를 증가시킨다.
* 새로운 문제를 생성한다.

---

## 오답 처리

* 입력 영역을 600ms 동안 빨간색으로 표시한다.
* Combo를 0으로 초기화한다.
* 입력을 초기화한다.
* 선택 위치를 처음 입력 영역으로 이동한다.
