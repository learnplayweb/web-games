# VS Code 개발 환경 구축

## 1. GitHub 저장소 클론

1. GitHub 저장소 열기
2. **`<> Code` (초록버튼)→ HTTPS** 선택
3. 저장소 주소 복사
4. VS Code 실행
5. `Ctrl + Shift + P`
6. `Git: Clone` 실행
7. 복사한 저장소 주소 붙여넣기
8. 저장 위치 선택 (예: `Documents/GitHub`)
9. 클론 완료 후 **Open Repository** 선택

프로젝트 예시

```text
Documents/
└── GitHub/
    └── web-games/
```
