# 보상 시스템 

## 문제 정답 보상

단계별 최고 별점에 따라 문제당 획득 골드를 차등 적용한다.
문제를 맞힐 때마다 현재 단계의 최고 별점 기준 보상 골드를 즉시 지급한다.
```
●	이전 최고 별 없음 : \+10 Gold

●	이전 최고 ★ : \+7 Gold

●	이전 최고 ★★ : \+4 Gold

●	이전 최고 ★★★ : \+1 Gold
```


## 콤보 보너스

* 콤보 보너스 = 최장 콤보 × COMBO_MULTIPLIER
* COMBO_MULTIPLIER는 상수로 관리한다.
* 초기값은 4로 설정한다.
* 밸런스 테스트 결과에 따라 상수만 변경하여 쉽게 조정할 수 있도록 구현한다.

---

## 별점 보상 계산

별점의 가치를 다음과 같이 정의한다.

* ★ = 10 Gold
* ★★ = 30 Gold
* ★★★ = 50 Gold

최초 도전 또는 최고 별점 갱신 시 지급 골드는

`새 최고 별 가치 − 이전 최고 별 가치`

로 계산한다.

예)

* 없음 → ★ = +10
* ★ → ★★ = +20
* ★★ → ★★★ = +20
* ★ → ★★★ = +40

별점 유지 또는 하락 시 추가 골드는 지급하지 않는다.

---

## 최고 별점 저장

최고 별점은 게임 전체가 아니라 각 단계별로 독립 저장한다.

예)

* Clock Lv.1 : ★★★
* Clock Lv.2 : ★★
* Clock Lv.3 : ★
* Clock Lv.4 : 없음

향후 저장 데이터는 단계별 최고 별점을 기준으로 관리한다. 

## 예외: 최종 레벨 보상 규칙

목적

* 최고 난도 반복 학습 유도
* 일반 레벨과 보상 체계 차별화

■ 골드 보상

* 문제 정답: 일반 레벨과 동일
* 최장 콤보 보너스: 일반 레벨과 동일(감소 없음)

■ 별점

* 플레이 결과에 따라 ★~★★★ 표시
* 별점은 저장하지 않음
* 매 플레이마다 새로 계산

■ 별점 보상(매 플레이 지급)
★★★ : +50골드
★★ : +30골드
★ : +10골드
(미만: 보상 없음)

■ 특징

* 최고 별점 저장 없음
* 보상 감소 없음
* 어려운 단계를 반복해도 항상 동일한 기준으로 보상 지급
* 반복 학습을 자연스럽게 유도
