상점의 파츠 선택 UI를 다음과 같이 수정한다.

1. 파츠 슬롯을 누르면 모달을 표시한다.

2. 모달에는 다음을 표시한다.
- 선택한 파츠를 크게 표시
- 구매 버튼
- 적용 버튼(랜덤 박스 제외)

3. 버튼 구성
- 일반 파츠
  - [구매 💎100]
  - [적용 💎10]
- 랜덤 박스
  - [구매 💎○○]

4. 구매
- 보유 여부와 관계없이 항상 구매할 수 있다.
- 동일한 파츠를 여러 개 구매할 수 있다.
- 이미 보유한 파츠도 추가 구매 가능하다.

5. 적용
- 이미 보유한 파츠만 활성화한다.
- 적용하면 현재 캐릭터의 **머리**를 해당 파츠로 변경한다.
- 현재 머리와 동일한 파츠인 경우에는 버튼 대신 '적용 중' 상태를 표시한다.

6. 모달 닫기
- 모달 바깥 배경을 누르면 닫는다.
- 별도의 닫기(X) 버튼은 만들지 않는다.

현재 단계에서는 UI만 구현한다.
구매, 적용, 골드 차감 등 실제 기능은 구현하지 않는다.
